# GuitarBro Shopify Analytics Dashboard

## Project Overview

A comprehensive analytics dashboard for GuitarBro's Shopify store that provides insights into business performance, order patterns, and operational metrics.

## Key Features

### Order Analytics
- Total orders, revenue, and average order value tracking
- Order trends over time (daily, weekly, monthly views)
- Product-level performance analysis

### Payment Method Analysis
- COD (Cash on Delivery) vs Prepaid order comparison
- Conversion rates by payment method
- Revenue split and profitability analysis per payment type

### Logistics & Fulfillment
- Shipping performance metrics
- Delivery time analysis
- Carrier-wise performance tracking

### Geographic Insights
- City-wise sales distribution
- Top performing cities and regions
- Shipping zone analysis

### Return Rate Analytics
- Overall return rate tracking
- Return reasons categorization
- Product-wise return analysis
- COD vs Prepaid return rate comparison
